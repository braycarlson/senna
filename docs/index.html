senna Proof of concept

A League of Legends companion app: champion builds, rune pages, tier lists, and live match insight, built with Tauri, Rust, and Vue.

Source on GitHub

Champion builds

Items and skill order for every champion, ready before the game loads.

Rune pages

Import complete rune pages straight into the League client in one action.

Tier lists

Ranked by win rate, with movers highlighting what climbed or fell this patch.

Live match insight

A cockpit that reads champion select and the live game as it happens.

Loadout automation

Summoner spells, item sets, and runes pushed to the client automatically.

Local and private

Talks to the League Client API on your machine. No login, no external servers.

Built with Tauri, Rust, and Vue 3.
